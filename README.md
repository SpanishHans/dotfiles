# My Dotfiles Managed with Chezmoi

This repository contains my personal dotfiles, managed with [Chezmoi](https://www.chezmoi.io/). It helps ensure a consistent development environment across multiple machines.

## About Chezmoi

Chezmoi is a powerful tool for managing your dotfiles across multiple systems. It allows you to configure, version control, and apply your dotfiles easily, ensuring that they stay up-to-date on all your devices.

## Getting Started

### Prerequisites

- [Chezmoi](https://www.chezmoi.io/docs/install/) installed on your system.
- Git installed for version control.

### Clone the Repository

You can clone this repository using Chezmoi directly:

```bash
chezmoi init https://github.com/<your-username>/dotfiles.git
